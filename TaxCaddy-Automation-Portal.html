ⓘ How to generate a GitHub Personal Access Token
  1. Sign in to GitHub.
  2. Click your profile picture in the top-right corner.
  3. Go to Settings.
  4. In the left sidebar, click Developer settings.
  5. Click Personal access tokens.
  6. Select Tokens (classic).
  7. Click Generate new token (classic), set required scopes: repo workflow read:user
  8. Copy and paste the token below.
